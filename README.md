# 🛒 E-Commerce Sales Analytics Dashboard (Excel Project)

## 📌 Project Overview

This project presents an interactive E-Commerce Sales Analytics Dashboard built using Microsoft Excel. The dashboard is designed to analyze overall sales performance, profitability, customer trends, and operational efficiency through dynamic visuals and KPI indicators.

## 📈 Dashboard Insights



### 👥 Orders by Gender
Analyzes order distribution between male and female customers.

### 📅 Month-Wise Orders
Displays monthly sales trends to identify peak and low-performing periods.

### 🏷️ Category-Wise Profit
Shows profitability across product categories such as Home, Fashion, Electronics, Sports, and Beauty.

### 📦 Product-Wise Profit
Highlights top-performing products including iPhone, Samsung TV, Treadmill, and more.

### 💳 Payment Method Analysis
Breakdown of customer payment preferences:

- Cash on Delivery  
- Credit Card  
- Debit Card  
- Net Banking  
- UPI

## 🎯 Objective

The goal of this project is to transform raw e-commerce data into meaningful business insights using Excel. This dashboard helps in understanding revenue trends, customer behavior, product profitability, and operational performance in a visually structured and user-friendly format.
